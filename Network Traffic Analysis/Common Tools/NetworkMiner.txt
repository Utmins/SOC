NetworkMiner — это инструмент сетевого криминалистического анализа (NFAT) с открытым исходным кодом для Windows (но также работает в Linux / Mac OS X / FreeBSD).

https://www.netresec.com/?page=NetworkMiner

Особенности NetworkMiner:

	-	Его можно использовать как пассивный сетевой сниффер/инструмент захвата пакетов для обнаружения операционных систем, сеансов, имен хостов, открытых портов и т. д. без передачи трафика в сеть.
	-	Он анализировать файлы PCAP для офлайн-анализа и восстанавливать/собирать переданные файлы и сертификаты из файлов PCAP.
	-	Упрощает выполнение расширенного анализа сетевого трафика (NTA), предоставляя извлеченные артефакты в интуитивно понятном пользовательском интерфейсе.

Вкратце, это действие по записи пакетов сетевого трафика и созданию расследуемых источников и установлению анализа первопричины события.
Конечная цель — предоставить достаточно информации для обнаружения вредоносных действий, нарушений безопасности, соответствия политике/регулированию, работоспособности системы и поведения пользователя.

Для захвата надлежащего сетевого трафика требуются знания и инструменты.
Обычно есть только один шанс собрать живой трафик в качестве доказательства.
Существует множество ресурсов для сбора данных сетевой криминалистики:

	-	TAPS
	-	InLine Devices
	-	SPAN Ports
	-	Hubs
	-	Switches
	-	Routers
	-	DHCP Servers
	-	Name Servers
	-	Authentication Servers
	-	Firewalls
	-	Web Proxies
	-	Central Log Servers
	-	Logs (IDS/IPS, Application, OS, Device

NetworkMiner обладает обширным перечнем возможностей.
Вот некоторые (так сказать основные) из них:

	Возможность				Описание
	Traffic sniffing			Он может перехватывать трафик, прослушивать его, собирать и регистрировать пакеты, проходящие через сеть.
	Parsing PCAP files			Он может анализировать файлы pcap и подробно показывать содержимое пакетов.
	Protocol analysis			Он может определять используемые протоколы из проанализированного файла pcap.
	OS fingerprinting			Он может определять используемую ОС, считывая файл pcap.
						Эта функция в значительной степени опирается на Satori и p0f.
	File Extraction				Он может извлекать изображения, файлы HTML и электронные письма из проанализированного файла pcap.
	Credential grabbing			Он может извлекать учетные данные из проанализированного файла pcap.
	Clear text keyword parsing		Он может извлекать ключевые слова и строки открытого текста из проанализированного файла pcap

NetworkMiner работает в 2х режимах:

	A)	Sniffer Mode
		Хотя он имеет функцию сниффинга, он не предназначен для использования в качестве сниффера.
		Функция сниффинга доступна только в Windows.
		Однако остальные функции доступны в ОС Windows и Linux.
		Исходя из опыта, функция сниффинга не так надежна, как другие функции.
		Поэтому мы рекомендуем не использовать этот инструмент в качестве основного сниффера.
		Даже в официальном описании инструмента упоминается, что этот инструмент является «инструментом анализа сетевой криминалистики», но его можно использовать как «сниффер».
		Другими словами, это инструмент анализа сетевой криминалистики с функцией сниффера, но он не является специализированным сниффером, как Wireshark и tcpdump

	B)	Packet Parsing/Processing
		NetworkMiner может анализировать захваты трафика, чтобы иметь быстрый обзор и информацию о исследованном захвате.
		Этот режим работы в основном предлагается для захвата "low hanging fruit" перед погружением в более глубокое расследование

Преимущества и Недостатки NetworkMiner

	) Pros
		Снятие отпечатков ОС
		Простое извлечение файлов
		Получение учетных данных
		Анализ ключевых слов в открытом тексте
		Общий обзор
	
	) Cons
		Не полезен при активном сниффинге
		Не полезен для большого исследования pcap
		Ограниченная фильтрация
		Не создан для ручного исследования трафика


************************
* NetworkMiner Toolbar *
************************

	1)	File Menu
		Оно поможет вам загрузить файл Pcap или получить Pcap по IP.
		Вы также можете перетаскивать файлы pcap.
		NetworkMiner также может получать Pcaps по IP.

	2)	Tools Menu
		Меню инструментов поможет вам очистить панель управления и удалить собранные данные.

	3)	Help Menu
		Панель кейсов показывает список исследованных pcap-файлов.
		Вы можете перезагрузить/обновить, просмотреть метаданные и удалить загруженные файлы из этой панели.

***************************
* NetworkMiner Case Panel *
***************************

Данная панель расположена в правой части рабочего окна

Панель кейсов показывает список исследованных pcap-файлов. 
Вы можете перезагрузить/обновить, просмотреть метаданные и удалить загруженные файлы из этой панели.

*********************
* NetworkMiner TABs *
*********************

	a)	Hosts
		Меню «hosts» показывает идентифицированные хосты в файле pcap.
		В этом разделе представлена ​​информация о:

			-	IP-адресе
			-	MAC-адресе
			-	Типе ОС
			-	Открытых портах
			-	Отправленных/полученных пакетах
			-	Входящих/исходящих сеансах
			-	Сведениях о хосте

		Для идентификации ОС используются репозитории Satori GitHub и p0f, а база данных MAC-адресов использует репозиторий mac-ages GitHub.

		Вы можете отсортировать идентифицированные хосты с помощью меню сортировки.
		Вы также можете изменить цвет хостов.
		Некоторые функции (поиск OSINT) доступны только в премиум-режиме.
		Меню правой кнопки мыши также помогает вам скопировать выбранное значение.

	b)	Files
		Меню файла показывает извлеченные файлы из исследованных pcaps.
		В этом разделе представлена ​​информация о:

			-	Frame number
			-	Filename
			-	Extension
			-	Size
			-	Source and destination address
			-	Source and destination port
			-	Protocol
			-	Timestamp
			-	Reconstructed path
			-	Details

		Некоторые функции (поиск хэша OSINT и отправка образцов) доступны только в премиум-режиме.
		Здесь также доступна строка поиска.
		В этой части также полезно контекстное меню.
		Вы можете легко открывать файлы и папки и просматривать подробные сведения о файлах.

	c)	Images
		Меню файла показывает извлеченные изображения из исследованных pcaps.
		Меню правой кнопки мыши также полезно в этой части.
		Вы можете открывать файлы и легко увеличивать и уменьшать масштаб

		При наведении курсора на изображение отображается подробная информация о файле (адрес источника и назначения, а также путь к файлу).

	d)	Messages
		Меню сообщений показывает извлеченные письма, чаты и сообщения из исследованных pcaps.
		В этом разделе представлена ​​информация о:

			-	Frame number
			-	Source and destination host 
			-	Protocol
			-	Sender (From)
			-	Receiver (To)
			-	Timestamp
			-	Sizeе

		После того, как вы отфильтруете трафик и получите совпадение, вы обнаружите дополнительные сведения, такие как вложения и атрибуты выбранного сообщения.
		Обратите внимание, что здесь также доступна строка поиска.
		Здесь доступно контекстное меню.
		Вы можете использовать встроенный просмотрщик для изучения общей информации и опцию «открыть файл» для изучения вложений.		

	e)	Credentials
		Меню учетных данных показывает извлеченные учетные данные и хэши паролей из исследованных pcaps.
		Вы можете использовать Hashcat (GitHub) и John the Ripper (GitHub) для расшифровки извлеченных учетных данных.
		NetworkMiner может извлекать учетные данные, включая:

			-	Kerberos hashes
			-	NTLM hashes
			-	RDP cookies
			-	HTTP cookies
			-	HTTP requests
			-	IMAP
			-	FTP
			-	SMTP
			-	MS SQLL

		Меню правой кнопки мыши также полезно в этой части.
		Вы можете легко скопировать значения имени пользователя и пароля.

	f)	Sessions
		Меню сеанса показывает обнаруженные сеансы в файле pcap.
		Этот раздел содержит информацию о:

			-	Frame number
			-	Client and server address
			-	Source and destination port
			-	Protocol
			-	Start timeа

		Вы можете искать ключевые слова внутри фреймов с помощью панели фильтрации.
		Также можно фильтровать определенные столбцы меню сеанса.
		Это меню принимает четыре типа входных данных:

			-	"ExactPhrase"
			-	"AllWords"
			-	"AnyWord"
			-	"RegExe"

	g)	DNS
		Меню DNS показывает DNS-запросы с подробностями.
		В этом разделе представлена ​​информация о:

			-	Frame number
			-	Timestamp
			-	Client and server
			-	Source and destination port 
			-	IP TTL
			-	DNS time
			-	Transaction ID and type
			-	DNS query and answer
			-	Alexa Top 1M

		Некоторые функции (Alexa Top 1M) доступны только в премиум-режиме
		Здесь также доступна строка поиска

	h)	Parameters
		Меню файла показывает извлеченные параметры из исследованных pcaps.
		Этот раздел содержит информацию о:

			-	Parameter name
			-	Parameter value
			-	Frame number
			-	Source and destination host
			-	Source and destination port
			-	Timestamp
			-	Detail

		В этой части также полезно контекстное меню.
		Вы можете легко скопировать параметры и значения.

	i)	Keywords
		Меню файла показывает извлеченные ключевые слова из исследованных pcaps. В этом разделе представлена ​​информация о:

			-	Frame number
			-	Timestamp
			-	Keyword
			-	Context
			-	Source and destination host
			-	Source and destination port

		Как фильтровать ключевые слова;

			-	Add keywords
			-	Reload case files

		Вы можете фильтровать несколько ключевых слов в этом разделе; однако вы должны перезагрузить файлы дел после обновления ключевых слов поиска.
		Поиск ключевых слов исследует все возможные данные в обработанных pcaps.

	j)	Anomalies
		Меню аномалий показывает обнаруженные аномалии в обработанном pcap.
		Обратите внимание, что NetworkMiner не обозначен как IDS.
		Однако разработчики добавили некоторые обнаружения для эксплойта EternalBlue и попыток спуфинга.
