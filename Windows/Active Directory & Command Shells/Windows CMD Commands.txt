* CMD *

	++++++++++++++++++++
	+ General Commands +
	++++++++++++++++++++

	-	systeminfo
		Отображает информацию о конфигурации операционной системы для локального или удаленного компьютера

	-	hostname
		Отображает имя текущего хоста.

	-	ver
		Отображает текущую версию Windows.

	-	page up (кнопка на клавиатуре)
		Помещает в командную строку первую команду из истории сеансов.

	-	page down (кнопка на клавиатуре)
		Помещает в командную строку посленюю команду из истории сеансов.

	-	F3 (кнопка на клавиатуре)
		Повторно вводит всю предыдущую запись в командную строку.

	-	F5 (кнопка на клавиатуре)
		Нажатие F5 несколько раз позволяет нам циклически просматривать предыдущие команды.

	-	F7 (кнопка на клавиатуре)
		Открывает интерактивный список предыдущих команд.

	-	F9 (кнопка на клавиатуре)
		Вводит команду в нашу подсказку на основе указанного числа. Цифра соответствует месту команды в нашей истории.

	-	dir
		Перечисляет содержимое каталога.

			dir /A <attributes>
			Перечисляет содержимое каталога с указанными атрибутами.
	
				dir /A:H	-	Перечисляет скрытые файлы в текущем каталоге.
				dir /A:R	-	Перечисляет файлы, доступные только для чтения, в текущем каталоге.
	
	-	cd 
		Отображает текущий рабочий каталог.
	
			cd <path>
			Перемещается с текущего каталога в указанныйю.
	
	-	tree
		Графически (в виде дерева) отображает структуру каталогов по указанному пути.
	
			tree /F <path>
			Графически (в виде дерева) отображает структуру каталогов по указанному пути, включая файлы внутри каталога.
	
	-	cls
		Очищает терминал.
	
	-	mkdir (or md) <directory name>
		Создает каталог в текущем рабочем каталоге (или указанном каталоге) с указанным именем.
	
	-	rmdir (or rd) <key> <directory name>
		Удаляет каталог в текущем рабочем каталоге (или указанном каталоге) с указанным именем.
	
			/S			-	Полностью удаляет все каталоги и файлы в указанном каталоге.
	
	-	move [source] [destination]
		Перемещает файл(ы) из исходной папки в папку назначения.
	
	-	copy [source] [destination] <key>
		Копирует файл(ы) из исходной папки в папку назначения.
		Работает только с файлами, а не папками.
	
			/V			-	Проверяет правильность копирования файла или файлов.
	
	-	xcopy <key> [source] [destination]
		Cкопирует файлы и папки из исходной папки в папку назначения
		***	Заменен Robocopy и в настоящее время устарел.
	
			/E			-	Копирует файлы и папки из исходной папки в папку назначения, включая пустые каталоги.
			/K			-	Копирует файлы и папки из исходной папки в папку назначения.
							Сохраняет текущие атрибуты скопированных файлов.
	
	-	robocopy <key> [source] [destination]
		Копирует файлы и папки из исходной папки в папку назначения.
		Он имеет более надежный набор функций по сравнению с xcopy.
	
			/E /MIR /A-:SH		-	Копирует файлы и папки из исходной папки в папку назначения.
									Отражает каталог назначения в исходный и очищает все дополнительные атрибуты с помощью параметра /A-:SH.
	
	-	more <key> <file>
		Отображает вывод файла или команды по одному экрану за раз.
	
			/S			-	Отображает вывод файла или команды по одному экрану за раз. Сжимает несколько пустых строк в одну.
	
	-	type <file>
		Отображает содержание файла
	
	-	echo "example string"
		Отображает указанные в ковычках данные на экране

			echo "example string" > <filename>
			Записывает предоставленное содержимое в новый или существующий файл с указанным именем.
			Если файл не существует, будет создан новый; в противном случае содержимое предыдущего файла будет перезаписано.
	
			echo "example string" >> <filename>
			Добавляет предоставленное содержимое в существующий файл.
	
	-	ren <filename1> <filename2>
		Переименовывает файл.
	
	-	del <key> <file>
		Удаляет файл или файлы.
	
			/A:R		-	Удаляет файл или файлы с установленным атрибутом «только для чтения».
			/A:H		-	Удаляет файл или файлы с установленным скрытым атрибутом.
	
	-	erase <file1> <file2> <file3>
		Удаляет файл или файлы.
		Взаимозаменяема с командой del.

	++++++++++++++++++++++++
	+ File System Commands +
	++++++++++++++++++++++++

	-	fsutil
		Это мощная системная утилита командной строки Windows, предназначенная для управления файловой системой, томами, точками монтирования, символьными ссылками, файлами и кластерами.
		***	Требуются права администратора для большинства операций
	
			fsutil file createNew <filename> <length>
			используется для создания нового пустого файла заданного размера (в байтах) в Windows через командную строку.

	+++++++++++++++++++++++++
	+ Find & Filter Content +
	+++++++++++++++++++++++++

	-	where <file>
		Отображает расположение предоставленных файла(-ов).
	
			where /R <working directory> <file>
			Детально ищет предоставленные файлы, начиная с указанного каталога.
	
	-	find
		Используется для поиска строк, содержащих определённый текст, в файле или выводе другой команды	
	
			find "example string" <file>
			Ищет текстовую строку в файле или файлах и отображает строки текста, содержащие указанную строку.
	
			find /i "<looking element>"
			Ищет совпадения с искомым элементом. Используется как в работе с файлами, так и с выводом на экран.
	
			find /i "<looking word>" < <file_name>
			Ищет совпадение в указанном файле и если находит, то выводит всю строку (до ВВОДА), даже если она разделена другими знаками препинания.
	
			find /N /I /V "key-word" <file_name>
			Отобразит все строки, в которых нет "key-word", при этом так же отобрахит номера строк и проигнориреут регистр букв.
	
	-	findstr
		Ищет шаблоны текста в файлах
	
	-	comp <file1> <file2>
		Побайтно сравнивает содержимое двух файлов или наборов файлов.
	
	-	fc <file1> <file2> /N
		Сравнивает два файла или набор файлов и отображает различия между ними нумеруя строки
	
	-	sort
		Считывает входные данные, сортирует данные и записывает результаты на экран, в файл или другое устройство.

	++++++++++++++++++++
	+ Network Commands +
	++++++++++++++++++++

	-	ipconfig
		Просматривает базовые сетевые конфигурации.

			ipconfig /?
			Отображает справку и информацию об использовании ipconfig.

			ipconfig /all
			Просмотр подробной информации о конфигурации сети.

	-	net
		Утилита CLI, содержащая несколько команд для управления и настройки сетевых ресурсов.

			net share
			Отображает информацию обо всех ресурсах, которые являются общими на локальном компьютере.

			net view
			Отображает список доменов, компьютеров или ресурсов, совместно используемых указанным компьютером.

	-	arp
		Отображает и управляет содержимым и записями в кэше протокола разрешения адресов (ARP).

			arp /a
			Отображает содержимое и записи, содержащиеся в кэше протокола разрешения адресов (ARP)
	
	-	route
		Используется для просмотра, добавления и удаления маршрутов в таблице маршрутизации

			route print
			Показать текущую таблицу маршрутизации

	-	netstat <key>
		Утиилита, которая используется для отображения сетевых подключений, таблицы маршрутизации и статистики сетевого интерфейса.
	
			netstat - an
			Отображает все активные сетевые подключения и слушающих портов в числовом формате, без попытки разрешить DNS-имена хостов..

	-	nslookup <IP address/URL>
		Запрашивает DNS имя или адрес.

	-	whoami
		Отображает имя пользователя, вошедшего в систему в данный момент.

			whoami /priv
			Отображает привилегии безопасности текущего пользователя.

			whoami /groups
			Отображает группы пользователей, к которым принадлежит текущий пользователь.

			whoami /all
			Отображает всю информацию о текущем пользователе, включая имя пользователя, идентификаторы безопасности (SID), привилегии и группы.

	-	net user
		Отображает список учетных записей пользователей на компьютере.

	-	net localgroup
		Отображает имя сервера и названия локальных групп на компьютере.

	-	net group
		Отображает имя сервера и имена групп на сервере.
		Можно использовать только в том случае, если машина присоединена к домену.

			net group "Domain Admins" /domain
			Отображает всех пользователей, входящих в группу "Domain Admins" в Active Directory-домене

	++++++++++++
	+ Services +
	++++++++++++

	-	sc <key>
		Перечисляет все запущенные службы и предоставляет дополнительную информацию для каждой службы.

			sc query <Name>
			Перечисляет сведения о конкретной службе по имени.
	
			sc query type= service
			Перечисляет, какие службы в настоящее время активно работают в системе.
			Расстояние между необязательными параметрами запроса имеет решающее значение.
			Например, type= service, type=service и type =service — это совершенно разные способы размещения этого параметра.
	
			sc start <Name>
			Запускает службу по имени.
	
			sc stop <Name>
			Останавливает службу по имени.
	
			sc config <Name> start=disabled
			Изменяет настройки указанной службы.
	
	-	tasklist <key>
		Используется в командной строке Windows (CMD) для отображения списка всех запущенных процессов на локальном (или удалённом) компьютере.
	
			tasklist /svc
			Предоставляет список служб, работающих под каждым процессом в системе.
	
	-	net start
		Перечисляет все запущенные службы.
	
	-	wmic service List brief
		Перечисляет все службы в системе, использующие WMIC.
		Включает такую информацию, как ExitCode, Name, ProcessID, StartMode, State и Status.

	+++++++++++++++++++
	+ Scheduled Tasks +
	+++++++++++++++++++

	-	schtasks <key>
		Отображает все задачи, запланированные на локальном компьютере.

			schtasks /query
			Отображает все задачи, запланированные на локальном компьютере. Взаимозаменяема с командой schtasks.

			schtasks /query /V /FO list
			Отображает все запланированные задачи с подробной информацией в формате списка.

			schtasks /create
			Позволяет создавать запланированные задачи.

			schtasks /create /sc <Schedule Frequency> /tn <Task Name> /tr <Program Path>
			Создает новую запланированную задачу на основе выбранного расписания с указанным именем и программой, указанной для запуска при запуске задачи.

			schtasks /change
			Позволяет изменять существующую запланированную задачу.

			schtasks /change /tn <Task Name> /ru <Username> /rp <Password>
			Изменяет запланированную задачу с указанным именем для запуска с разрешениями учетной записи пользователя, используя предоставленный пароль для аутентификации.

			schtasks /delete
			Позволяет удалять запланированные задачи.

			schtasks /delete /tn <Task Name>
			Удаляет запланированную задачу с соответствующим именем.

	++++++++++++++++++++++
	+ Event Log Commands +
	++++++++++++++++++++++

	-	wevtutil <key>
		Это мощная консольная утилита в Windows, предназначенная для управления журналами событий Windows (Event Logs).
		
			wevtutil el
			Использует утилиту командной строки событий Windows для перечисления всех источников журналов.

			wevtutil gl "name"
			Соберет информацию о конфигурации для указанного журнала.

			wevtutil qe <Name> /c:5 /rd:true /f:text
			Запрашивает журнал событий.

			wevtutil epl <Name> C:\system_export.evtx
			Экспортирует журнал.

	+++++++++++++++++++++
	+ Registry Commands +
	+++++++++++++++++++++

	-	REG <command> <key>
		Это встроенная консольная утилита в Windows, предназначенная для управления реестром Windows (Registry) из командной строки или .BAT-скриптов.

			REG QUERY <HIVE>\PATH\KEY
			Использует reg.exe для запроса реестра.
	
			REG QUERY <HIVE> /F "Password" /t REG_SZ /S /K
			Ищет определенные строки в кусте реестра.
	
			REG add "<HIVE>\PATH\KEY" /v access /t REG_SZ /d "C:\Users\htb-student\Downloads\payload.exe"
			Использует Reg.exe для создания новой пары ключ/значение.

	+++++++++++++++++++++++++
	+ Environment Variables +
	+++++++++++++++++++++++++

		-	%EXAMPLE_VARIABLE%
			Пример формата переменной среды.

		-	set
			Отображает все доступные переменные среды в системе.

				set <%VARIABLE_NAME%>
				Отобразит значение указанной переменной среды.
				Его также можно использовать для установки значения переменной.

				set <%VARIABLE_NAME%>= <Value>
				Создает новую переменную среды или изменяет существующую и устанавливает значение для текущего сеанса командной строки.

				setx <%VARIABLE_NAME%> <Value>
				Создает новую переменную среды или изменяет существующую и устанавливает значение глобально, внося изменения в реестр.

				set <%VARIABLE_NAME%>=
				Удаляет переменную среды с указанным именем для текущего сеанса командной строки.

				setx <%VARIABLE_NAME%> ""
				Удаляет переменную среды с указанным именем глобально.
		
		-	echo <%VARIABLE_NAME%>
			Распечатывает значение указанной переменной среды. Он не может вносить какие-либо изменения в переменные и только выводит значения на консоль.

	++++++++++++++++++++++++++
	+ Input/Output Operators +
	++++++++++++++++++++++++++

	-	[command] > [file]
		Перенаправляет вывод команды в файл, при этом содержимое указанных файлов перезаписывается.
	
	-	[command] >> [file]
		Перенаправляет вывод команды в файл. Добавляет дополнительный вывод, не перезаписывая исходное содержимое файла.
	
	-	[command] < [file]
		Перенаправляет выходные данные файла и передает их команде.
	
	-	[command] | [command2]
		Выходные данные первой команды обрабатываются второй командой посредством использования |.
	
	-	[command] & [command2]
		Выполняет обе команды последовательно. Он не выполняет проверки, чтобы определить, прошла ли какая-либо команда или нет.
	
	-	[command] && [command2]
		Проверяет, успешно ли выполняется первая команда, а затем выполняет вторую команду. Если первая команда завершается неудачей, выполнение текущей команды останавливается, а вторая команда не выполняется.
	
	-	[command] || [command2]
		Проверяет, не удалось ли выполнить первую команду успешно, и если да, то переходит к выполнению второй команды.
